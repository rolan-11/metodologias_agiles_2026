# Ejercicio Nro: 6

## Enunciado
Brindar un ejemplo en que se vean reflejadas las siguientes ventajas de la aplicación de normas en el desarrollo de software (Mejora calidad, Optimiza proceso, Mejora satisfacción, Reduce costos, Aumenta competitividad).
Ayuda: Utiliza las referencias de las normas enunciadas anteriormente: ISO, CMMI, IEEE, ITIL.

## Resolución
**Ejemplo: Desarrollo de un Tablero de Ventas (Dashboard) para una cadena comercial.**

Imaginemos un equipo que desarrolla un dashboard web utilizando .NET y SQL Server. Al aplicar normas internacionales, el impacto es el siguiente:

* **Mejora la calidad (ISO/IEC 25000):** Al aplicar esta norma de calidad, se estandarizan las pruebas sobre la base de datos. Esto garantiza que el sistema sea seguro, no se caiga cuando hay muchos usuarios conectados y los datos sean confiables
* **Optimiza el proceso (CMMI):** Al tener un nivel de madurez alto en sus procesos, el equipo organiza mejor las tareas y los tiempos. Los programadores no pierden tiempo rehaciendo codigo porque las reglas de desarrollo ya están claras desde el principio
* **Mejora la satisfacción del cliente (IEEE):** Utilizando el estándar IEEE 830 para documentar los requisitos el equipo deja por escrito y sin ambigüedades exactamente lo que el cliente necesita por ejemplo, ver rankings de vendedores por región) Al entregar justo lo que se pidió, el cliente queda satisfecho.
* **Reduce costos:** Gracias a las normas anteriores, los errores de lógica o de diseño se detectan en las primeras etapas. Arreglar un diagrama o un documento es muchísimo mas barato que tener que reestructurar todo el código cuando el sistema ya está instalado en los locales
* **Aumenta la competitividad (ITIL):** Al usar las buenas practicas de ITIL para organizar el soporte tecnico y el mantenimiento la empresa creadora del software ofrece respuestas rápidas si el sistema falla. Esto les da una ventaja enorme frente a la competencia que no tiene un buen servicio post-venta
